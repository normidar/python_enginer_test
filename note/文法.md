### システム関数

#### システム関数ではない関数

```
count() のシステム関数は存在しない（これはlistの関数で、アイテムを数える用、使い方：list.count(obj)）

calculate() はありません
number() はありません
mod() はありません
```

#### システム関数である関数

```
list(列)　リスト化関数。
enumerate(リスト)　リターンされる物はセットの列： set(index,item)
dir(sys)　のようにインスタンスのメンバーを表す。
```

#### 文字の比較

```
ASCIIコードに変換して比較しています。

数字 < 大文字のアルファベット < 小文字のアルファベット
0 < 9 < A < Z < a < z 
```

```
r をstrの前に付くとstrはそのまま表示される。
例えば：
print('a\na') #２行に表示される
print(r'a\na') #　a\na　のまま出力される
```
